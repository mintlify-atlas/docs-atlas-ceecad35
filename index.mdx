Claude Subconscious gives Claude Code a long-term memory. A [Letta](https://letta.com) agent runs silently in the background, reading your code, learning your patterns, and surfacing context before each prompt. Claude Code forgets everything between sessions — Subconscious remembers.

## What Subconscious does

After each Claude Code response, the session transcript is sent to a Letta agent running in the background. The agent reads your files, updates its memory, and whispers guidance back before your next prompt — all without adding latency to your workflow.

<CardGroup cols={3}>
  <Card title="Watches sessions" icon="eye">
    Every transcript is sent to the agent after each response via the `Stop` hook.
  </Card>
  <Card title="Reads your code" icon="magnifying-glass">
    The agent uses `Read`, `Grep`, and `Glob` to explore your codebase while processing.
  </Card>
  <Card title="Builds memory" icon="database">
    Eight memory blocks accumulate preferences, project context, and patterns over time.
  </Card>
  <Card title="Whispers back" icon="comment">
    Messages are injected before each prompt so Claude sees your agent's guidance.
  </Card>
  <Card title="Never blocks" icon="bolt">
    Async processing means the `Stop` hook exits immediately — no waiting.
  </Card>
  <Card title="Shares across projects" icon="share-nodes">
    One agent brain serves all your repositories with per-project conversation threads.
  </Card>
</CardGroup>

## Get started

<Steps>
  <Step title="Get a Letta API key">
    Sign up at [app.letta.com](https://app.letta.com) and copy your API key.
  </Step>
  <Step title="Install the plugin">
    Run these two commands inside Claude Code:
    ```
    /plugin marketplace add letta-ai/claude-subconscious
    /plugin install claude-subconscious@claude-subconscious
    ```
  </Step>
  <Step title="Set your API key">
    Export the key in your shell profile:
    ```bash
    export LETTA_API_KEY="your-api-key"
    ```
  </Step>
  <Step title="Start coding">
    Open any project in Claude Code. Subconscious activates automatically on the first session. After a few sessions it will have enough context to provide useful guidance.
  </Step>
</Steps>

<Note>
  The first few sessions are a ramp-up period. Subconscious starts with minimal context and gets smarter as it observes more sessions, reads your code, and learns your patterns.
</Note>
