## Prerequisites

- **Node.js 18+** — required to run the hook scripts
- **Claude Code** — the plugin hooks into Claude Code's lifecycle events
- **Letta account** — get your API key at [app.letta.com](https://app.letta.com)

## Install the plugin

<Tabs>
  <Tab title="From marketplace (recommended)">
    <Steps>
      <Step title="Add from marketplace">
        Run these two commands inside Claude Code:

        ```
        /plugin marketplace add letta-ai/claude-subconscious
        /plugin install claude-subconscious@claude-subconscious
        ```
      </Step>
      <Step title="Set your API key">
        Export your Letta API key in your shell:

        ```bash
        export LETTA_API_KEY="your-api-key"
        ```

        Get your key from [app.letta.com](https://app.letta.com). Add this line to your `~/.bashrc` or `~/.zshrc` to make it permanent.
      </Step>
      <Step title="Start a new session">
        Open a new Claude Code session. On first run, the plugin automatically imports the bundled Subconscious agent and saves its ID for future use. You'll see the agent name, ID, and a supervision link printed to the terminal.
      </Step>
    </Steps>
  </Tab>
  <Tab title="From source">
    <Steps>
      <Step title="Clone the repository">
        ```bash
        git clone https://github.com/letta-ai/claude-subconscious.git
        cd claude-subconscious
        npm install
        ```
      </Step>
      <Step title="Enable the plugin">
        From inside the cloned directory, run in Claude Code:

        ```
        /plugin enable .
        ```

        To enable globally for all projects:

        ```
        /plugin enable --global .
        ```

        If enabling from a different directory, use the full path to the cloned repo.
      </Step>
      <Step title="Set your API key">
        ```bash
        export LETTA_API_KEY="your-api-key"
        ```

        Get your key from [app.letta.com](https://app.letta.com). Add to your shell profile to make it permanent.
      </Step>
      <Step title="Start a new session">
        Open a new Claude Code session. The plugin will auto-import the bundled agent and display connection info on startup.
      </Step>
    </Steps>
  </Tab>
</Tabs>

## Required configuration

Only one environment variable is required:

```bash
export LETTA_API_KEY="your-api-key"
```

Everything else is handled automatically. The plugin will:

1. Query available models on your Letta server
2. Import the bundled Subconscious agent if no agent is configured
3. Save the agent ID to `~/.letta/claude-subconscious/config.json` for future sessions

For additional configuration options — modes, model overrides, self-hosted servers — see [Environment Variables](/configuration/environment-variables).

## First run expectations

<Info>
  On first use, the agent starts with minimal context. It takes a few sessions before it has enough signal to provide useful guidance. Give it time — it reads your code, learns your patterns, and gets smarter the more it observes.
</Info>

During the first session you should see a startup banner printed directly to your terminal (not captured by Claude) with:

- Agent name and ID
- Model, mode, and SDK tools settings
- A supervision link to `app.letta.com` (Letta Cloud only)

After that:

- A session start notification is sent to the Letta agent
- Memory and messages are injected into Claude's context before each prompt (may be empty at first)
- The transcript is sent to the agent asynchronously after each response

## Linux: tmpfs workaround

<Warning>
  If plugin installation fails with `EXDEV: cross-device link not permitted`, your `/tmp` is on a different filesystem than your home directory. This is common on Ubuntu, Fedora, and Arch Linux.
</Warning>

Set `TMPDIR` to a path on the same filesystem as your home directory:

```bash
mkdir -p ~/.claude/tmp
export TMPDIR="$HOME/.claude/tmp"
```

Add both lines to your `~/.bashrc` or `~/.zshrc` to make the fix permanent. This works around a [known Claude Code bug](https://github.com/anthropics/claude-code/issues/14799).
